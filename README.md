Descripcion general del sistema:
                                Este proyecto es un sistema de gestión y préstamos diseñado para administrar las operaciones diarias de una biblioteca

Instrucciones de ejecucion:
                            1) registrarse como usuario ingresando la informacion que se pide
                            2) en caso de buscar un lirbo para saber su informacion podes mirar el catalogo de libros que tiene el archivo o si ya conoces que libros hay directamente colocar el nombre
                            3) para pedir prestado un libro ingresar el dni y colocar el nombre del libro que lleva 
                            4) en caso de querer devolver el libro seleccionar la opcion de devolucion y si necesita devolver todo junto escribir "s" en caso contrario escribir "n" y dar el nombre del libro que va a devolver
                            5) para ver cual fue el libro mas pedido seleccionar la opcion 5
                            6) para saber cuantos prestamos tiene el usuario seleccionar la opcion 6
                            7) en caso de querer ver como funciona la funcion de calcular multa seleccionar la opcion 7 y elegir cuantos dias restar a la fecha de vencimiento para que se actualice (hay que hacer que la nueva fecha de vencimiento sea menor que la fecha actual para poder comprobar el funcionamiento) la fecha de vencimiento se la informa cuando hace el prestamo.
                            8) seleccione la opcion 8 en caso de no querer realizar mas acciones
